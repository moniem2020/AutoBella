# AutoBella - Mobile Car Washing & Detailing Services

Welcome to AutoBella, a modern car washing and detailing service that comes to you. We provide professional car washing and detailing services directly at your home or preferred location, offering convenience and quality care for your vehicle.

## About AutoBella

AutoBella is a mobile car services platform designed to bring professional car washing and detailing to your doorstep. Whether you need a quick wash or comprehensive detailing, our skilled professionals will handle your vehicle with care.

### Services Offered
- **Mobile Car Washing** - Professional washing at your location
- **Detailing Services** - Comprehensive interior and exterior detailing
- **Client Memberships** - Flexible membership plans for regular customers
- **B2B Memberships** - Commercial packages for businesses

## Getting Started

This is a [Next.js](https://nextjs.org) project. To run the development server:

```bash
bun dev
# or
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the website.

The website auto-updates as you edit files. Start by modifying files in the `src/app/` directory.

## Project Structure

- **`src/app/`** - Next.js app directory with pages for home, about, services, memberships, and contact
- **`src/components/`** - Reusable React components and page sections
- **`public/`** - Static assets and images

## Technology Stack

- [Next.js](https://nextjs.org) - React framework for production
- [TypeScript](https://www.typescriptlang.org/) - Type-safe JavaScript
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [shadcn/ui](https://ui.shadcn.com/) - High-quality UI components

## Development

Key commands:
